# SoundSteps Landing Page - README

## Project Overview

SoundSteps is a parent-led speech practice app supporting young children with speech development. This project involved designing and building a responsive landing page to introduce SoundSteps, explain the problem it addresses, and capture Early Access signups.

---

## Website Structure

### 1. Hero Section

* Displays logo prominently.
* Clear headline explaining the purpose.
* Supporting text.
* Primary CTA button: "Join Early Access".

### 2. The Problem Parents Face

* Describes common challenges for parents:

  * Long wait times for speech support.
  * Uncertainty on how to help at home.
  * Lack of structured tools for daily practice.
* Key statistics visually highlighted to build credibility.

### 3. Our Approach

* Introduces SoundSteps as a calm, structured tool for home practice.

### 4. How SoundSteps Works

Step flow illustrated with friendly icons/illustrations:

1. Listen to a sound.
2. Child repeats the sound.
3. Parent encourages.
4. Recordings saved.
5. Gentle progress suggestions.

### 5. Early Access Signup

* Prominent form capturing:

  * Parent/Carer name
  * Email
  * Location
  * How did you hear about us?
  * Child's age range
  * Who the app is for
* Support status options:

  * Waiting for NHS support
  * Receiving NHS support
  * Receiving private support
  * Not currently receiving support
* Additional insight: "What concerns you most about your child's speech development?"
* Optional questions about feedback willingness and local support availability.

### 6. Our Mission

* Explains why SoundSteps is being built and role of early families in shaping the product.

### 7. Footer

* Contact email.
* Company information.
* Privacy reassurance.

## Design Direction

* Tone: Warm, Supportive, Calm, Modern, Trustworthy.
* Avoid overly clinical or children's game visuals.
* Use rounded shapes, soft colour palette, friendly illustrations, clean modern typography.

### Suggested Illustrations

* Parent and child practising speech.
* Sound waves / speech bubbles.
* Microphone / recording symbol.
* Step-by-step progress concept.
* Encouragement / positive reinforcement.
* Style: Friendly, soft colours, slightly playful, flat or lightly textured vector style, matching the logo.
* Format: SVG preferred.
